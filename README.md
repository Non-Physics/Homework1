# 项目内容

这是在Fashion-MNIST数据集上训练的三层神经网络模型，通过使用Numpy完成模型推导。

## 环境设置

- Python 3.11
- NumPy
- Matplotlib (仅用于可视化)

## 如何训练

1. 克隆仓库：`git clone <repo_url>`
2. 进入仓库目录：`cd <repo_name>`
3. 运行训练脚本：`python train.py`

## 如何测试

1. 确保你已经下载了模型权重并放在了项目目录中。
2. 运行测试脚本：`python test.py`

## 模型权重下载
